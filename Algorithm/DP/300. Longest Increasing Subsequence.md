## Note
### Method 1: DP

#### Code
```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        dp = [1] * len(nums)

        for i in range(1, len(nums)):
            for j in range(i):
                if nums[i] > nums[j]:
                    dp[i] = max(dp[i], dp[j] + 1)
        return max(dp)
```
#### Analysis
##### Time Complexity: $O(n^2)$
##### Space Complexity: $O(n)$

#### Code
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def removeElements(self, head: Optional[ListNode], val: int) -> Optional[ListNode]:
        
        dummy_head = ListNode()
        dummy_head.next = head

        curr = head
        prev = dummy_head

        while curr:
            if curr.val == val:
                prev.next = curr.next
            else:
                prev = curr
                
            curr = curr.next

        return dummy_head.next
```
#### Analysis
##### Time Complexity: $O(n)$
##### Space Complexity: $O(1)$
